¿CÓMO ESTÁ ESTO MONTADO?

Entre que me hace ilu explicarlo, y que Dani me ha dicho que deje esto aquí escrito por si acaso, pues voy a explicar un poco cómo está todo esto montado.


EL HARDWARE

Es un equipo que literalmente, monté de piezas que fuí comprando de amigos a buen precio, más componentes nuevos que he ido poniéndole al equipo, y que ahora mismo está en mi salón encendido 24/7, conectado al router por Ethernet.

Yo y mis servidores.

Procedo ahora a detallar los componentes del equipo a fecha de hoy:

COMPONENTE NOMBRE
CPU Intel Core 2 Quad Q6600 @ 2.40 GHz
GPU Nvidia GeForce GTX 750 2GB GDDR5
RAM 8GB DDR2 @ 800 MHz
PLACA BASE Gigabyte P35-DS3L
DISCO(S) DURO(S) 1 x 120GB SSD, 2 x 500GB HDD
FUENTE DE ALIMENTACIÓN 650W

EL SOFTWARE

Principalmente, todo esto funciona gracias a XAMPP, una distribución de Apache gratuita y fácil de usar, que además integra MySQL, PHP, FileZilla, entre otras cosas. Hay activos un Servidor Web (el que te permite leer este mismo texto), una base de datos MySQL, y un servidor FTP.

Finalmente, este servidor, alojado en el equipo descrito previamente en la sección de Hardware, se están redirigiendo los puertos pertinentes para el Servidor Web, y el servidor de Filezilla Server, y están accesibles desde la Internet pública.

Se está utilizando también NO-IP como servicio de DDNS. (Dynamic Domain Name System). Es un servicio gratuito de resolución de nombres (se creéis que yo tengo el dinero para pagar un dominio y un servicio de hosting) que permite acceder a esta web utilizando la URL que has utilizado para acceder aquí.

Y la verdad, no hay mucho más que explicar, creo que he sido bastante claro.
